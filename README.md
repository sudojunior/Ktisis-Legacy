# Ktisis-Legacy
[![discord](https://img.shields.io/discord/975894364020686878)](https://discord.gg/kUG3W8B8Ny)

Looking for the Ktisis plugin? Check it out [here](https://github.com/ktisis-tools/Ktisis)!

This is an **archived repo** of Ktisis' v0.2 and v0.3 versions and commit histories as of 07/14/2026. This archive is purely available for research and development purposes and is **not** usable as a Dalamud plugin. This repo will **not** be updated with any changes in XIV patches or Dalamud API bumps.
